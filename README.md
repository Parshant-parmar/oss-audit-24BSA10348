# Open Source Audit Project

Name: Parshant
Reg Number:24BSA10348

Software Chosen: Git

## Scripts

1. Script 1 - System Identity Report
Displays system details like OS, kernel, user, uptime.

2. Script 2 - Package Inspector
Checks if Git is installed and shows version.

3. Script 3 - Disk and Permission Auditor
Shows directory permissions and size.

4. Script 4 - Log File Analyzer
Counts occurrences of a keyword in log files.

5. Script 5 - Manifest Generator
Generates a personalized open-source manifesto.

## How to Run

chmod +x script1.sh
./script1.sh

chmod +x script2.sh
./script2.sh

chmod +x script3.sh
./script3.sh

chmod +x script4.sh
./script4.sh /var/log/system.log error

chmod +x script5.sh
./script5.sh
